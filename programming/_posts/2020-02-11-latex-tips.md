### まとめ
#### 定理の名前
```

```

#### tilder
```
\sim 
%=> ~

\tilde
%=> 上付き ~
```

#### 数列空間
$ \ell^2 $

小文字lの筆記体は以下のように書く．
```
\ell
```

#### 集合包含
$ A \subsetneq B $: 真部分集合
```
\subsetneq
```

#### 二項係数(組み合わせ)
$ \binom{n}{k} $: 縦ベクトルの表示
```
\binom{above}{below}
```

#### 斜体など
イタリック
```
\textit{イタリックにしたいアルファベット}
```
